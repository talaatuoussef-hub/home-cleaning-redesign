# Home Cleaning Service Website Redesign

## Project Overview
This project focuses on redesigning a home cleaning service website to improve usability, visual clarity, and the overall booking experience.

The original website had several UX issues such as weak visual hierarchy, cluttered sections, and unclear booking actions. The redesign aims to transform the interface into a clearer, more structured digital product that guides users smoothly through the service discovery and booking process.

## Problem
During the analysis of the original website, several usability problems were identified:

• Weak visual hierarchy  
• Cluttered layout and inconsistent spacing  
• Important actions like booking were not clearly highlighted  
• Low color contrast (#4A94F7 on white background)  
• Service information was not well structured  

These issues made it harder for users to quickly understand the services and complete a booking.

## Solution
The redesign focused on improving clarity, structure, and usability.

Key improvements include:

• A clear hero section with a strong booking call-to-action  
• Improved content hierarchy and spacing  
• Structured service sections for easier scanning  
• A simplified “How It Works” flow  
• A more organized pricing layout  
• Updated brand color from **#4A94F7** to **#0063E8** to improve contrast and accessibility  

The goal was to create a cleaner interface that helps users quickly understand the service and take action.

## Design Process
The design process included the following steps:

1. UX Analysis of the existing website  
2. Identifying usability problems and friction points  
3. Information hierarchy restructuring  
4. UI redesign with improved layout and typography  
5. Designing clearer service sections and booking flow
